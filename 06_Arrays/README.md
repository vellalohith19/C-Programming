# Arrays

## Overview

This folder contains C programs that demonstrate the fundamentals of one-dimensional arrays. Arrays are used to store multiple values of the same data type in contiguous memory locations and are widely used in C programming, Embedded Systems, and IoT applications.

## Programs

1. Store And Display Array
2. Sum Of Array Elements
3. Largest Element
4. Smallest Element
5. Average Of Array
6. Reverse Array
7. Copy Array
8. Sort Array Ascending
9. Sort Array Descending
10. Search Element

## Concepts Covered

* One-dimensional arrays
* Array declaration and initialization
* Array indexing
* Array traversal
* Input and output using arrays
* Searching
* Sorting
* Element copying
* Aggregate operations (sum, average, maximum, minimum)

## Learning Outcomes

After completing these programs, you will be able to:

* Declare and use one-dimensional arrays.
* Store and retrieve array elements.
* Traverse arrays using loops.
* Perform common array operations.
* Implement basic searching and sorting algorithms.
* Build a strong foundation for advanced data structures.
