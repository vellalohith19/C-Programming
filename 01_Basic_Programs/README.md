# Basic Programs

## Overview

This folder contains beginner-level C programs that introduce the fundamental concepts of the C programming language. These programs provide a strong foundation for learning programming by focusing on basic input/output operations, program structure, formatted output, and character handling.

## Programs

1. Hello World
2. Print Name
3. Bio Data
4. Formatted Output
5. Character Output

## Concepts Covered

* Structure of a C program
* Header files (`#include <stdio.h>`)
* `main()` function
* `printf()` function
* Formatted console output
* Character data type (`char`)
* Escape sequences (`\n`)
* Basic console output

## Folder Structure

```text
01_Basic_Programs/
├── 01_Hello_World/
├── 02_Print_Name/
├── 03_Bio_Data/
├── 04_Formatted_Output/
└── 05_Character_Output/
```

## Learning Outcomes

After completing these programs, you will be able to:

* Understand the basic structure of a C program.
* Display text using `printf()`.
* Create formatted console output.
* Use escape sequences for better readability.
* Declare and print character variables.
* Write, compile, and execute simple C programs.
