# Unions

## Overview

This folder contains C programs that demonstrate the concept of **unions**. A union is a user-defined data type that allows multiple members to share the same memory location. Unlike structures, only one member of a union can hold a valid value at a time. Unions are widely used in embedded systems, device drivers, communication protocols, and memory-constrained applications where efficient memory usage is important.
