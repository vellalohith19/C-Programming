# Conditional Statements

## Overview

This folder contains C programs that demonstrate decision-making using conditional statements. These programs help in understanding how a program can execute different blocks of code based on specific conditions.

## Programs

1. Even or Odd
2. Positive or Negative
3. Vowel or Consonant
4. Largest of Two Numbers
5. Largest of Three Numbers
6. Smallest of Three Numbers
7. Leap Year
8. Character Type
9. Grade Calculation
10. Simple Calculator

## Concepts Covered

* `if`
* `if-else`
* `else if`
* `switch`
* Relational operators
* Logical operators (`&&`, `||`)
* Decision making
* Character comparison

## Learning Outcomes

After completing these programs, you will be able to:

* Make decisions using `if`, `else if`, and `switch`.
* Compare numbers and characters.
* Use logical and relational operators.
* Build simple interactive programs.
