# Functions

## Overview

This folder contains C programs that demonstrate the use of functions in C programming. Functions help organize code into reusable blocks, making programs easier to read, maintain, and debug. This section also introduces recursion, one of the fundamental concepts in programming.

## Programs

1. Function Without Arguments Without Return Value
2. Function With Arguments Without Return Value
3. Function Without Arguments With Return Value
4. Function With Arguments With Return Value
5. Factorial Using Function
6. Prime Number Using Function
7. Fibonacci Using Function
8. Recursive Factorial
9. Recursive Fibonacci
10. Power Using Function

## Concepts Covered

* Function declaration (Prototype)
* Function definition
* Function call
* Function arguments (Parameters)
* Return values
* `void` functions
* Recursive functions
* Code reusability
* Modular programming

## Learning Outcomes

After completing these programs, you will be able to:

* Understand the purpose of functions in C.
* Create and call user-defined functions.
* Pass arguments to functions.
* Return values from functions.
* Write modular and reusable programs.
* Understand the basics of recursion.
* Improve code readability and maintainability.
