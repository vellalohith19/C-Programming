# Operators

This folder contains C programs that demonstrate arithmetic operators, user input, and mathematical calculations.

## Programs

1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Quotient and Remainder
6. Swap Two Numbers
7. Simple Interest
8. Area of Circle
9. Celsius to Fahrenheit
10. Fahrenheit to Celsius

## Concepts Covered

* Variables
* Arithmetic operators (`+`, `-`, `*`, `/`, `%`)
* User input using `scanf()`
* Output using `printf()`
* Integer and floating-point data types
* Mathematical expressions
